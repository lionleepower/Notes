## Obsidian 最常用快捷键  
  
Windows / Linux 常用：  
  
| 操作 | 快捷键 |  
|---|---|  
| 新建笔记 | `Ctrl + N` |  
| 快速打开笔记 | `Ctrl + O` |  
| 打开命令面板 | `Ctrl + P` |  
| 全局搜索 | `Ctrl + Shift + F` |  
| 当前页搜索 | `Ctrl + F` |  
| 加粗 | `Ctrl + B` |  
| 斜体 | `Ctrl + I` |  
| 插入外部链接 | `Ctrl + K` |  
| 返回上一位置 | `Ctrl + Alt + ←` |  
| 前进到下一位置 | `Ctrl + Alt + →` |  
  
Mac 上通常把 `Ctrl` 换成 `Cmd`。  
  
Obsidian 官方帮助里也说明，Hotkeys 可以在 `Settings → Hotkeys` 里查看、修改和自定义；最快查看某个命令快捷键的方法，是在命令面板里搜索这个命令。:contentReference[oaicite:1]{index=1}  
  
## Obsidian 常用指令怎么学  
  
最重要的是这个：  
  
```text  
Ctrl + P  
```  
  
它是 Obsidian 的“总控制台”。  
  
你按下 `Ctrl + P` 后，可以搜索：  
  
```text  
New note  
Open graph view  
Open today's daily note  
Insert template  
Toggle reading view  
Move file to another folder  
```  
  
所以你不需要一开始背所有快捷键。你只要记住：  
  
```text  
Ctrl + P = 找任何命令  
Ctrl + O = 找任何笔记  
Ctrl + Shift + F = 搜索所有内容  
```  
  
这三个先熟练，Obsidian 就能顺起来。
