给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。  
找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。  
返回容器可以储存的最大水量。  
**说明：**你不能倾斜容器。

```
class Solution(object):
    def maxArea(self, height):
        """
        :type height: List[int]
        :rtype: int
        """
        ans =0
        left=0
        right = len(height)-1
        while left \< right:
            area = (right - left) * min(height[right], height[left])
            ans = max(area,ans)
            if height[left]\<height[right]:
                left +=1
            else:
                right-=1
        return ans
        
```

```
class Solution(object):
    def lengthOfLongestSubstring(self, s):
        """
        :type s: str
        :rtype: int
        """
        ans = 0
        left = 0
        cnt = Counter() #hashmap
        for right, x in enumerate(s):
            cnt[x] += 1
            while cnt[x] \>1:
                cnt[s[left]] -= 1
                left += 1
            ans = max(right - left +1, ans)
        return ans
          
```
 
==为什么一定要减== ==cnt[s[left]]====？==  
因为滑动窗口 [left, right] 缩小时，真正被“踢出去”的是最左边那个字符。  
比如字符串：
 
s = "abba"  
我们走到 right = 2 时，窗口是 "abb"。  
这时：
 
cnt['b'] = 2  
说明有重复，要缩窗口。  
窗口左边 left = 0，对应字符是 'a'，所以先移出的是 'a'，不是 'b'。  
也就是说应该：
 
cnt['a'] -= 1  
left += 1  
如果还重复，再继续移。  
然后下一次 left = 1，这时移出的才是 'b'。  
所以缩窗口的逻辑永远是：
 
cnt[s[left]] -= 1  
left += 1  
不是直接对 x 动手。
